SMAC WEB

Bienvenido

SMAC WEB es una plataforma avanzada de monitoreo en tiempo real que permite visualizar datos clave del vehículo, como la posición del acelerador, RPM y el estado de los sensores. Esta herramienta proporciona retroalimentación detallada para optimizar el rendimiento del vehículo y prolongar su vida útil, al mismo tiempo que ayuda al usuario a mejorar sus prácticas de conducción.

  • Monitorear los datos críticos del vehículo.
  • Proporcionar retroalimentación sobre los hábitos de conducción.
  • Facilitar el mantenimiento proactivo del vehículo.
  • Prolongar la vida util de las piezas mecanicas de los vehiculos.
  • Contribuir en la seguridad vial al prolongar la vida util del automovil.

SMAC WEB permite la recopilación, análisis y visualización de datos en tiempo real.

SMAC WEB utiliza una ESP32 conectada al sistema OBD2 del vehículo, que permite la comunicación y recolección de datos a través del protocolo CAN. La interfaz web muestra los datos recopilados para que el usuario pueda observar una retroalimenatcion en tiempo real.
